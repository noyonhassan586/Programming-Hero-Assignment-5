Question-1: What are the different ways to select an element in the DOM?

Ans:
getElementById: Selects an element by its ID.
getElementsByClassName: Selects all elements with a specific class name.
getElementsByTagName: Selects all elements with a specific tag name.
querySelector: Selects the first element that matches a CSS selector.
querySelectorAll: Selects all elements that match a CSS selector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Ans:
innerHTML: Gets or sets the HTML inside an element, including tags.
innerText: Gets or sets the text inside an element, ignoring the HTML tags and spaces.
textContent: Similar to innerText, but gets or sets the text including hidden text in the element.

Question-3: What is event delegation in the DOM?

Ans:
Event delegation in the DOM: Event delegation is a way to handle events on multiple child elements by adding a single event listener to their parent. The event listener catches the event when it bubbles up from the child elements.

Question-4: What is event bubbling in the DOM?

Ans:
Event bubbling is the process where an event starts from the target element and "bubbles up" to the parent elements in the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

Ans:
Create: Use document.createElement('tagName') to create a new element.
Add: Use appendChild() or insertBefore() to add an element to the DOM.
Remove: Use removeChild() to remove an element from its parent.
